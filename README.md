# WeatherProject-OpenAPI

### 📖 프로젝트 개요
WeatherProject-OpenAPI는 공공 API를 사용하여 날씨 정보를 제공하는 웹으로, 사용자는 국내의 다양한 도시의 현재 날씨와 기상 정보를 조회할 수 있습니다.<br/><br/>

### 🚀 프로젝트 목표
- 국내 도시의 실시간 날씨 정보를 제공
- 사용자가 쉽게 날씨 정보를 조회할 수 있는 기능 구현

- ## 🗺 기술 설계도
(여기에 기술 설계도를 추가할 수 있습니다)

## 📝 프로젝트 설명

### 💼 WeatherProject-OpenAPI의 기능
1. **실시간 날씨 정보 조회**
   - 사용자 입력을 통해 도시별 실시간 날씨 정보 제공
2. **다양한 도시의 기상 데이터 제공**
   - 여러 도시의 날씨 정보를 한눈에 비교 가능
3. **사용자 친화적인 인터페이스**
   - 직관적인 UI를 통해 손쉽게 날씨 정보 접근

### 📦 설치 방법
1. 저장소를 클론합니다:
    ```sh
    git clone https://github.com/buseonghyeon/WeatherProject-OpenAPI.git
    cd WeatherProject-OpenAPI
    ```

2. 필요한 Gradle 의존성을 설치합니다:
    ```sh
    ./gradlew build
    ```

### 실행

1. 애플리케이션을 실행합니다:
    ```sh
    ./gradlew bootRun
    ```
